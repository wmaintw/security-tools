Step 1. Install and start Docker

Step 2. Run the following command to add execution permission to shell script

  sudo chmod +x run-dvwa-docker-container.sh

Step 3. Run shell script to run DVWA

  ./run-dvwa-docker-container.sh

Step4. Visit DVWA via http://0.0.0.0:9099/dvwa, the default login credential is: admin/password
